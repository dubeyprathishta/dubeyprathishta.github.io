CSS Selectors

Applying CSS to Different Parts of HTML

  1. The element selector targets based on their HTML tag name.
  2. Class selectors targets elements based on the value of the class attribute.
  3. ID selectors target elements based on the value of the id attribute.
  4. Attribute selectors target elements based on their attributes and values .
  5. The universal selector targets all elements.

Explanation

  • tag elements are specified with the html tag elements before flower braces.
  • class selector are specified with a dot"." before the class attribute value with no spaces endong with a flower braces just like another styling elements. The class name can be same for different elements of the html.
  • ID selector firstly the value of the ID attribute cannot be same with different elements the ID vlaue should be unique for every other element.The ID selector can be specified in the style.css file with the "#" pound sign before the ID value.
  • Attribute selector can be worked on both ID and class selectors.The attribute selector is specified within a squarebractes"[]"after the class or ID value.
  • Universal selector are nothing but which is applied to every element of the html irrespective of claas,ID,attrinbute.It is specified with asterick " * " symbol before the flower braces.
